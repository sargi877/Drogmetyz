# Drogmetyz — Інструкція для власника

Вітаємо! Ваш новий сайт Drogmetyz готовий. Цей файл є інструкцією для вас, як ним керувати.

## 1. Як оновити наявність та ціни
Ваш Google Sheet є "адмін-панеллю". Щоб оновити ціну або наявність:
1. Відкрийте ваш Google Sheet.
2. Знайдіть потрібну деталь (за ID, діаметром чи довжиною).
3. Змініть значення у колонці `Наявність` або `Ціна (грн)`.
   - Рекомендовані значення для наявності: `В наявності`, `Обмежено`, `Немає`. 
   - Якщо поле залишити пустим, сайт автоматично покаже `Уточнюйте`.
4. Зміни з'являться на сайті автоматично через кілька хвилин (кеш діє 5 хвилин).

## 2. Як додати нові товари або категорії
- Щоб додати розмір для існуючого стандарту, просто додайте рядок у Google Sheet з тими ж значеннями `Категорія` та `Назва / Стандарт`.
- Щоб додати нову категорію, яка зараз має статус "Незабаром" (наприклад, "Гайки"):
  1. Додайте товари цієї категорії у ваш Google Sheet (колонка Категорія має бути "Гайки").
  2. В коді сайту у файлі `assets/js/views/catalog.js` видаліть `aria-disabled="true"` та `style="opacity: 0.5;"` з блоку "Гайки", та перетворіть `div` на посилання `<a href="#/catalog/haiky" class="card">...</a>`.
  
## 3. Налаштування Telegram / Google Form
Зараз кнопка "Оформити бронювання" використовує заглушки. Коли ви створите свою Google форму:
1. Відкрийте файл `assets/js/reservation.js`.
2. Замініть значення `RESERVATION_FORM_BASE_URL` на посилання вашої форми.
3. Замініть значення в об'єкті `FORM_FIELD_IDS` на реальні `entry.XXXXXX` ідентифікатори з вашої форми для імені та списку товарів.
*Поки ви цього не зробили, функція "Скопіювати список" працює як надійна альтернатива, що дозволяє клієнтам скопіювати замовлення текстом.*

## 4. Як замінити контактні дані
У файлі `index.html` є спеціальні мітки `TODO(owner)`. Відкрийте його у текстовому редакторі і знайдіть їх, щоб:
- Вписати свій реальний номер телефону замість `+380 XX XXX XX XX`.
- Вписати реальну адресу.
- Оновити години роботи.

## 5. Як замінити посилання на таблицю (Джерело даних)
1. У вашій таблиці Google Sheet оберіть `Файл -> Поділитися -> Опублікувати в інтернеті`.
2. Оберіть аркуш і формат **CSV**. Скопіюйте посилання.
3. Відкрийте файл `assets/js/data.js` і замініть значення `PASTE_PUBLISHED_CSV_URL_HERE` на скопійоване посилання.

## 6. Як розгорнути сайт
Сайт не потребує спеціальної збірки (build). Він складається зі звичайних файлів HTML/CSS/JS.
1. Завантажте цю папку як репозиторій на GitHub.
2. Зайдіть у налаштування репозиторію (Settings) -> Pages.
3. У розділі "Build and deployment" виберіть Source: `Deploy from a branch` і вкажіть вашу головну гілку (наприклад, `main`).
4. Натисніть Save. Ваш сайт запрацює за кілька хвилин!
